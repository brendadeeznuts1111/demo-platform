CSS Parser API

Parser API

The Parser API allows developers to extend CSS with custom properties and parsing logic.

// Custom Property Registration Example CSS.registerProperty({ name: '--custom-spacing', syntax: '', inherits: false, initialValue: '0px' }); CSS.registerProperty({ name: '--custom-color', syntax: '', inherits: false, initialValue: 'transparent' }); CSS.registerProperty({ name: '--custom-size', syntax: ' | ', inherits: false, initialValue: '0px' }); // Usage in CSS .custom-element { padding: var(--custom-spacing); background-color: var(--custom-color); width: var(--custom-size); height: var(--custom-size); }

Typed OM API

CSS Typed OM API

Typed OM

The Typed Object Model API provides type-safe access to CSS values and enables mathematical operations.

// Typed OM Usage Example const element = document.querySelector('.element'); // Get typed values const width = element.attributeStyleMap.get('width'); const height = element.attributeStyleMap.get('height'); // Mathematical operations const totalWidth = CSS.px(100).add(CSS.em(2)); const doubleHeight = height.mul(2); // Set typed values element.attributeStyleMap.set('width', totalWidth); element.attributeStyleMap.set('height', doubleHeight); // Unit conversion const inches = CSS.px(96).to('in'); // 1in const centimeters = CSS.px(96).to('cm'); // 2.54cm // Complex calculations const margin = CSS.px(10).add(CSS.percent(5)); const padding = CSS.px(20).max(CSS.em(1));
